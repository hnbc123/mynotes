# IDEA常用快捷键

* 复制光标所在行或复制选定内容

  > Ctrl + D

* 删除行

  > Ctrl + Y

* 大小写转换

  > Ctrl + Shift + U

* 前进/后退

  > Ctrl + Alt + ← / →

* 查看历史记录

  > Ctrl + E

* 自动补全代码

  自动补全代码尾部的分号、括号对等等，还可以自动格式化完成的代码。

  > Ctrl + Shift + Enter

* 快速抽取变量

  > Ctrl + Alt + V
  >
  > 或者使用后缀补全（语句末加.var）

* 重命名

  > Shift + F6

* 跳转到指定的Java文件

  > Ctrl + N

* 用if、while等围绕选中的代码行

  > Ctrl + Alt + T

* 选择光标所在处的区域

  > Ctrl + W
  >
  > Ctrl + Shift + W (反选)

* 跳转到出错语句

  > F2 (跳转到上一个)
  >
  > Shift + F2 (跳转到下一个)

* 自动去除无用的import语句

  > Ctrl + Alt + O

* 显示当前文件结构

  > Ctrl + F12

* 自动生成代码

  > Alt + Enter

* 格式化代码

  > Ctrl + Shift + L
